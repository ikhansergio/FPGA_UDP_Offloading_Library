AXISx8_RGMII_BRIDGE - Verilog library implementing 1000Base-T/100Base-T/10Base-T Ethernet PHY interfacing via RGMII interface.

# AXISx8_RMII_BRIDGE [[...]](./AXISx8_RMII_BRIDGE/README.md)

AXISx8_RMII_BRIDGE - Verilog library implementing 100Base-T/10Base-T Ethernet PHY interfacing via RMII interface.

# AXISx8_UDP_Offloading_Engine [[...]](./AXISx8_UDP_Offloading_Engine/README.md)

AXISx8_UDP_Offloading_Engine - Verilog library that implements the reception of Ethernet II packets, calculation of their CRC, MAC address control, IPv4 address control, UDP port control and control of checksums. ARP and ICMP PING functions are also implemented.

# AXISx8_UDP_Framing_AXISx32_Sink [[...]](./AXISx8_UDP_Framing_AXISx32_Sink/README.md)

AXISx8_UDP_Framing_AXISx32_Sink - Verilog library that implements UDP datagram generation from an AXISx32 stream. IP automatically adds the MAC header, IP4 header, and UDP header, and calculates all necessary checksums.
